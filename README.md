# V2raySocks
## 一个售卖v2ray的whmcs插件
* 当前版本 0.0.1Beta3

## 支持的功能：
* 流量图表(已修复自动任务过于频繁导致记录过多的问题)
* 流量重置(每月开始，每月底，由结算日计算)
* 公告信息(已修复HTML问题)
* VMESS二维码(已更新IOS/Mac和Win/Android)
* 随机UUID
* tls,混淆已支持
* 重置UUID

## 注意事项
